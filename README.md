# 🤖 React AI Chatbot

This is a simple AI chatbot built using React and Gemini API.

## 🚀 Tech Stack

- React JS
- useState Hook
- Fetch API
- Gemini API (Free Tier)

## ✨ Features

- User input handling
- Dynamic question & answer rendering
- API integration
- Basic chat UI


## 🛠 How to Run

1. Clone this repository
2. Install dependencies:
   npm install
3. Add your own Gemini API key inside:
   src/assets/constants.js
4. Run the project:
   npm run dev

## 📌 Note

This project uses Gemini Free Tier API.
If API limit is reached, generate your own API key.
